## Potato OS v0.7.0

### 1. Gemma 4 support

Potato OS now includes **experimental Gemma 4 support**.

- Supported Gemma 4 paths include `E2B`, `E4B`, and an experimental `26B-A4B` route
- On supported devices, Gemma 4 now runs on the faster `ik_llama` runtime path
- This is still an **experimental** feature, and on the current `ik_llama` path Gemma 4 is **text-only for now**

### 2. Fixes, improvements, and architecture changes

This release also includes a large batch of polish and platform work:

- The app platform was strengthened with app isolation, Chat as a first-class built-in app, and the new icon-rail shell layout
- Runtime behavior improved with better projector handling, active runtime-family display, and clearer model-loading progress
- OTA update flows improved with canonical release notes and a built-in "What's new" experience after updates
